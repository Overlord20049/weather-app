# Weather App (Python)

A simple command-line weather application that fetches real-time weather data using an API.

## Features
- Get current temperature
- Weather condition (e.g., Haze, Clear)
- Humidity information
- User input for any city

## Tech Stack
- Python
- Requests library
- API integration

## How to Run
1. Install dependencies:
   pip install requests

2. Run:
   python3 main.py

3. Enter city name

## Example Output
Weather in Delhi:
Temperature: 26°C  
Condition: Haze  
Humidity: 45%
